AI Meeting Notes Generator

Automates meeting documentation by converting raw transcripts into structured summaries, decisions, and action items using local AI (Ollama + n8n).

📌 Overview

This project builds an end-to-end AI automation pipeline that converts unstructured meeting text into structured insights using local LLM (Ollama) and workflow automation (n8n).

📊 Impact

Eliminates manual meeting notes
Improves decision tracking
Automates action item extraction
Reduces productivity overhead

🏗️ Workflow

Postman → Webhook (n8n) → Ollama LLM → JavaScript → Google Sheets

✨ Features

Webhook API trigger
Local AI processing (Ollama)
Structured summarization
Action item extraction
Owner & deadline detection
Google Sheets integration

🚀 Demo

Input

John will finish login page by Friday. Sarah confirmed API is ready. Mark will review designs tomorrow.

Output

✔ Summary generated
✔ Action items extracted
✔ Owners assigned
✔ Deadlines identified

⚙️ Tech Stack

n8n Ollama JavaScript Google Sheets Postman

💡 Why This Project

Most teams waste time writing meeting notes manually. This system automates the entire process using local AI while keeping data private and secure.
