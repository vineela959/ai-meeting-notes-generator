AI Meeting Notes Generator

End-to-end AI automation system that converts raw meeting transcripts into structured summaries, decisions, and action items using local LLM (Ollama) + n8n workflow automation.

View Project

📌 Project Overview

This project automates meeting documentation using a fully self-hosted AI pipeline. It removes manual note-taking by extracting structured insights from raw conversations and storing them in Google Sheets.

🏗️ System Architecture

Postman → Webhook (n8n) → Ollama LLM → JavaScript Parser → Google Sheets

✨ Key Features

Webhook-based API trigger
Local LLM processing (Ollama)
Structured meeting summarization
Action item extraction
Owner & deadline detection
Automated Google Sheets storage

⚙️ Tech Stack

n8n Ollama JavaScript Google Sheets Postman

🔄 Workflow

  • User sends meeting transcript via API (Postman)
  • Webhook receives data in n8n
  • Ollama processes and extracts insights
  • JavaScript formats structured output
  • Google Sheets stores final results

📊 Project Impact

  • Reduces manual meeting documentation effort
  • Improves clarity of action items
  • Automates workflow end-to-end
  • Ensures structured decision tracking

🔒 Privacy

This system uses a local LLM (Ollama), ensuring all data remains on-device. No external AI APIs are used.
